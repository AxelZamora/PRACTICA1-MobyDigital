ROADMAP

Que aprenderás

Durante el pre-academy aprenderás paso a paso las siguientes tecnologias para desenvolverte en el mundo profesional.

Git & GitHub

Aprende a trabajar versionando tu código de una forma segura y en equipo.

HTML5

Aprende HTML desde cero, sin conocimientos previos. Conoce todo sobre etiquetas HTML, semántica, y maquetado.

CSS3 & Boostrap

Aprende como dar estilos a los elementos de tu sitio web, Flexbox, Grid, Animaciones, modelo de caja y mucho más.

Javascript

Un lenguaje de programación interpretado. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Typescript

Superconjunto de JavaScript, que esencialmente añade tipos estáticos y objetos basados en clases.

Angular

Angular es un framework para desarrollo de aplicaciones de todo tipo, por defecto genera aplicaciones web muy poderosas y robustas
